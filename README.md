هکسو اسکنر یک ابزار بسیار پیشرفته و بهینه برای اسکن و بررسی کیفیت آی‌پی‌های کلادفلر (Cloudflare CDN IPs) است. این ابزار به صورت خودکار پینگ و کیفیت اتصال آی‌پی‌ها را بررسی کرده و بهترین و کم‌تاخیرترین آی‌پی‌ها را برای شما فیلتر می‌کند تا بتوانید در ابزارهایی مانند v2rayNG و سایر نرم‌افزارهای عبور از فیلترینگ استفاده نمایید.
